## School/Class Page

Create a new page accessible only to users in the Teacher role.

The page should include:
- A school dropdown populated from the `Schools` table.
- An `Add` button beside the school dropdown that opens a popup to create a new school.
- A class dropdown populated from the `Classes` table and filtered by the selected school.
- An `Add` button beside the class dropdown that opens a popup to create a new class for the selected school.
- A student grid filtered by the selected class.
- A search bar that filters the student grid by user display name.
