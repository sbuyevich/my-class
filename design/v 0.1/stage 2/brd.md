# My Class MVP Plan. Stage 2.

## Post Login 
No Login item in Navigation menu. 
App is opened with Login page as modal form. 

### Student Use Case 
Student can login or register if no record in Students table. 
Student redirect to Home page if login was successful.

### Teacher Use Case
Teacher login with credentials from appsettings.json and redirect to Home page.

## Students page
- add search bar on top of grid to filter students by name or last name
- Add remove button/action to remove student from database. it will force student to register again.
