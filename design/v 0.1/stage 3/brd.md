# Quiz

## Student Use Case
- Student see Quiz Answer page in menu after login
- Quiz Answer page has 4 big buttons with 1,2,3,4 caption
- Student click on button to submit answer

## Teacher Use Case
- see Quiz page in Teacher section

## Quiz Page
- Quiz page presents: 
    - quiz questions as one jpg image
    - timeout counted 
    - finish and next buttons
    - Student grid with status of answer for each active student

## Quiz Configuration
- Quiz is defined in folder pointed in appsettings.json
- root folder has json file with quiz description(TBD) 
- each question defines in sub-folders with jpg file as content and json as metadata (TBD)
