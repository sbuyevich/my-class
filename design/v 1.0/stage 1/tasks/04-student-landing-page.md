# Task 04 - Student Landing Page

## Goal

Build a simple student landing page shown after student login.

## Work

- Create the `/student` page.
- Show a clear waiting message telling the student to wait until the teacher is ready for the quiz.
- Restrict page access to students.
- Keep the page connected to the current class context.

## Deliverables

- Student landing page UI.
- Student-only access behavior.

## Acceptance Criteria

- Student can access `/student`.
- Teacher and unauthenticated users cannot use `/student`.
- Student login redirects to `/student`.
- The page displays a waiting message.
- The existing student quiz flow remains accessible from its intended navigation or direct route.

## Notes

- Do not change quiz behavior in this task.
